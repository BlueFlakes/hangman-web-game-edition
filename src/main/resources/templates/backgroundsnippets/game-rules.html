Hangman rules


There are 194 countries in the world. Do you know their capitals? Let's check!
We draw one for you. All letters have been converted to dashes.

If you write the wrong letter, you lose 1 point.
If you guess - dash changes for letter in the right place.

If you write the wrong word, you lose 2 points.
If you guess - dashes change for world.

If you have last health point, we give you a hint - name of the country.

Don't let the guy hang!!

Try to guess the word as soon as possible. The clock is ticking ;)
